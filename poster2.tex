\splitcolh{0.3\columnwidth}{20in}{
  \rulebox{Background}{
    \emph{Plasticine} a vector Coarse-Grained Reconfigurable Array:
    \begin{itemize}
        \item 6-stage, 16-lane 32-bit floating point SIMD pipelines
        \item Distributed 256-kByte memories
        \item DRAM controllers with tile load and scatter-gather support
    \end{itemize}
    \emph{Plasticine} demonstrated up to 95x speedup vs. an FPGA, and 77 times performance per Watt.
  }
